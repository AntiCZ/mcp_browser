BrowserMCP Enhanced Settings

Code Execution Mode

Safe Mode (Default): Code runs with limited, sandboxed API. No access to cookies, Chrome APIs, or network requests.

Unsafe Mode: Full access to window, document, fetch, and all browser APIs. Use with extreme caution!
⚠️ Security Warning: Unsafe mode allows executed code to:
  • Access all cookies and local storage
  • Make network requests to any domain
  • Access Chrome extension APIs
  • Modify any page content without restrictions
  • Potentially leak sensitive data
Only enable this if you fully trust the code source!
